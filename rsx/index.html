Regroove Synthesizer

RSX (Regroove Synthesizer) is an evolving synthesizer framework that is targeting portability and exchange between different devices running the engine. RS1 is the first implementation and is available for Android, Korg Logue, Move (Schwung), Windows and Linux as DAW plugins and MIDI-enabled standalone applications.
