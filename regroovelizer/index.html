Regroovelizer (Logue runtime)

Regroovelizer is a reverse engineered Logue runtime that is compatible with NTS-1 mkII and NTS-3 Logue oscillator and effect units. It was mostly created to provide an alternative test platform without having to use the Korg Kontrol Editor tooling. It is functional and I would like to continue development, as at the moment this only supports Arm32-capable platforms.
